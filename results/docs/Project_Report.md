# OFDM Wireless System Simulation Report

## Objective
Implement an OFDM-based wireless communication system and evaluate BER performance.

## Methodology
1. Random bit generation
2. QPSK modulation
3. OFDM modulation
4. AWGN channel transmission
5. OFDM demodulation
6. BER calculation

## Results
BER decreases with increasing SNR.

## Conclusion
OFDM provides robust transmission performance under noisy channel conditions.
